RootGraph v1.0 es un software educativo diseñado para facilitar el aprendizaje de los estudiantes que cursan la asignatura de Métodos 
Numéricos. Su principal función es graficar ecuaciones no lineales y calcular de manera aproximada la raíz de dichas ecuaciones. 
Estos procedimientos son fundamentales en el análisis numérico, ya que permiten encontrar soluciones a problemas matemáticos 
que no se resuelven fácilmente de forma algebraica.
